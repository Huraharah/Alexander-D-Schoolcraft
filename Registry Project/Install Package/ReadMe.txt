---Installation Instructions---
To install the software, simply extract the ZIP file to a location of your choosing.

This application requires Java Runtime, at least version 1.8. To download it, go to:

https://www.java.com/download/ie_manual.jsp

If you have a JDK or JSE at least version 8, you already have this version of the Java Runtime Environment.

Once extracted, you can launch the application from the LaunchUGR.exe application.
